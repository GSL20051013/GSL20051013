MA1112 Algebra - Reference Sheet

Part 1: Vector Algebra

1. Fundamental Concepts

  • Vector: A quantity with both magnitude and direction.
  • Scalar: A quantity with only magnitude.
  • Position Vector: A vector that starts from the origin O.
  • Unit Vector: A vector with a magnitude of 1. The unit vector in the direction of vector a is denoted as â and is calculated as:
    â = a / |a|
  • Magnitude of a Vector: For a = a₁i + a₂j + a₃k, the magnitude is:
    |a| = √(a₁² + a₂² + a₃²)
  • Direction Cosines: For a vector a = a₁i + a₂j + a₃k, the direction cosines are:
    cos α = a₁ / |a|, cos β = a₂ / |a|, cos γ = a₃ / |a|
    Property: cos²α + cos²β + cos²γ = 1

2. Vector Operations

  • Scalar (Dot) Product:
    a · b = |a| |b| cos θ
    a · b = a₁b₁ + a₂b₂ + a₃b₃
    If a ⊥ b, then a · b = 0.
    Projection of b onto a: b · â
  • Vector (Cross) Product:
    a × b = (|a| |b| sin θ) n̂
    a × b = (a₂b₃ - a₃b₂) i - (a₁b₃ - a₃b₁) j + (a₁b₂ - a₂b₁) k
    If a || b, then a × b = 0.
  • Scalar Triple Product:
    a · (b × c) = (a × b) · c
    Volume of a parallelepiped: V = |a · (b × c)|

3. Lines and Planes

  • Equation of a Line:
    Vector form: r = a + λv
    Cartesian form: (x - x₀)/a = (y - y₀)/b = (z - z₀)/c
  • Equation of a Plane:
    Vector form (point and two direction vectors): r = a + λb + μc
    Vector form (point and normal vector): (r - r₀) · n = 0
    Scalar form: ax + by + cz = d
  • Area of a Triangle: Given two vectors a and b forming two sides of a triangle, the area is:
    Area = ½ |a × b|
  • Skew Lines: Two lines are skew if they are not parallel and do not intersect. To check if two lines are skew:
    1. Check if their direction vectors are parallel.
    2. Check if they intersect by setting their vector equations equal to each other. If there is no solution, and they are not parallel, they are skew.

Part 2: Complex Numbers

1. Forms of Complex Numbers

  • Rectangular Form: z = x + iy
  • Polar Form: z = r(cos θ + i sin θ)
  • Exponential Form: z = reiθ

2. Basic Definitions

  • Modulus: |z| = r = √(x² + y²)
  • Argument: arg(z) = θ
  • Principal Argument: -π < Arg(z) ≤ π
  • Complex Conjugate: z̅ = x - iy
  • A number is purely imaginary if its real part is zero (Re(z) = 0).

3. Operations and Theorems

  • Euler's Identity: eiθ = cos θ + i sin θ
  • De Moivre's Theorem: (cos θ + i sin θ)ⁿ = cos(nθ) + i sin(nθ)
  • Triangle Inequality: |z₁ + z₂| ≤ |z₁| + |z₂|
  • |z₁ - z₂| ≥ ||z₁| - |z₂||

Part 3: Functions

1. Basic Concepts

  • Function: A rule that assigns to each element in the domain exactly one element in the co-domain.
  • Domain: The set of all possible input values.
  • Co-domain: The set of all possible output values.
  • Range: The set of all actual output values.

2. Types of Functions

  • One-to-one (Injective): Every element in the domain maps to a unique element in the range. To prove, show that if f(x₁) = f(x₂), then x₁ = x₂.
  • Onto (Surjective): The range is equal to the co-domain. Every element in the co-domain has a corresponding element in the domain.
  • Bijection: A function that is both one-to-one and onto.
  • Inverse Function (f⁻¹): A function that reverses the effect of f. An inverse function exists if and only if f is a bijection.
  • Even Function: f(-x) = f(x). The graph is symmetric about the y-axis.
  • Odd Function: f(-x) = -f(x). The graph is symmetric about the origin.

Part 4: Matrices

1. Basic Concepts

  • Matrix: A rectangular array of numbers.
  • Identity Matrix (I): A square matrix with 1s on the main diagonal and 0s elsewhere.
  • Transpose (Aᵀ): The matrix obtained by interchanging the rows and columns of A.

2. Determinants

  • Determinant (det(A) or |A|): A scalar value that can be computed from the elements of a square matrix.
  • A matrix is singular if det(A) = 0.
  • A matrix is non-singular or invertible if det(A) ≠ 0.
  • For a 2x2 matrix A = [[a, b], [c, d]], det(A) = ad - bc.

3. Inverse of a Matrix

  • Inverse (A⁻¹): A⁻¹ exists if and only if det(A) ≠ 0.
  • A A⁻¹ = A⁻¹ A = I
  • Formula for the inverse of a 2x2 matrix:
    If A = [[a, b], [c, d]], then A⁻¹ = (1/(ad-bc)) * [[d, -b], [-c, a]]
  • Finding the inverse of a larger matrix:
    1. Calculate the matrix of minors.
    2. Find the matrix of cofactors.
    3. Find the adjoint (or adjugate) of the matrix by transposing the cofactor matrix.
    4. The inverse is the adjoint matrix divided by the determinant.

Part 5: Limits and Series

1. Limits

  • Limit of a Function (ε-δ Definition): lim (x→c) f(x) = L if for every ε > 0, there exists a δ > 0 such that if 0 < |x - c| < δ, then |f(x) - L| < ε.
  • Limit of a Sequence (ε-n Definition): lim (n→∞) aₙ = L if for every ε > 0, there exists an integer N such that if n > N, then |aₙ - L| < ε.

2. Series

  • Convergent Series: A series is convergent if its sequence of partial sums converges.
  • p-series Test: The series Σ 1/nᵖ converges if p > 1 and diverges if p ≤ 1.
  • Comparison Test: If 0 ≤ aₙ ≤ bₙ for all n, then:
    • If Σ bₙ converges, then Σ aₙ converges.
    • If Σ an diverges, then Σ bₙ diverges.

Part 6: Numerical Methods

1. Newton-Raphson Method

  • An iterative method for finding successively better approximations to the roots (or zeroes) of a real-valued function.
  • Formula: xₙ₊₁ = xₙ - f(xₙ)/f'(xₙ)
